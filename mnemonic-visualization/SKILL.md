# 이미지 연상법 / 이중부호화 (Visualization / Dual Coding)

## 개요

단어의 뜻을 머릿속에 생생한 그림이나 동작으로 그려서 언어 정보(단어-뜻)와 시각 정보(이미지)를 동시에 인코딩한다. mnemonic-keyword가 "소리"에 초점을 맞춘다면, 이 기법은 "의미 자체의 시각화/동작화"에 초점을 맞춘다.

## 적합한 입력

- 동사 (동작을 그릴 수 있는 단어): 동작을 과장해서 묘사
- 공간/감각 형용사 (steep, blurry, vast 등)
- 한국어 발음 매칭이 잘 안 되는 구체 명사

부적합: 순수 추상 개념(justice, freedom 등)은 mnemonic-context나 mnemonic-storytelling이 더 적합.

## 절차

1. **핵심 의미 추출**: 단어의 핵심 뜻을 가장 구체적인 행동/장면으로 변환한다 (추상적 정의를 그대로 쓰지 않는다).
2. **과장/감각 디테일 추가**: 크기, 속도, 색깔, 소리 등을 과장해서 이미지를 더 강렬하게 만든다.
3. **동사의 경우 동작 시연**: 사용자가 직접 그 동작을 몸으로 흉내 낼 수 있도록 묘사한다.
4. (선택) mnemonic-keyword에서 만든 소리 연상이 있다면 같은 장면에 자연스럽게 결합한다.

## 출력 형식

```
**이미지 연상**
장면 묘사: <한두 문장의 생생한 시각/동작 묘사>
```

## 예시

- **plummet** (급락하다, 곤두박질치다)
  - 장면 묘사: 놀이공원의 자이로드롭이 굉음을 내며 순식간에 바닥까지 곤두박질치는 모습 — 그 속도와 심장이 철렁하는 느낌을 떠올린다.

- **steep** (가파른)
  - 장면 묘사: 거의 수직에 가까운 등산로를 두 손과 두 발을 다 써서 기어오르는 자신의 모습을 떠올린다.

## 주의

- 이미지는 짧고 한 장면으로 압축한다 — 여러 장면이 섞이면 오히려 혼란스럽다.
- 사용자에게 "이 장면을 3초간 눈을 감고 떠올려보세요"처럼 능동적 시각화를 유도하는 문구를 덧붙이면 효과가 커진다.
