# 음성·리듬 연상법 (Phonetic / Rhythm / Shadowing)

## 개요

소리 내어 반복하고, 리듬이나 강세 패턴을 과장해서 익히면 단어/문장의 발음과 형태가 청각·운동 기억으로 함께 저장된다. 특히 발음이 어렵거나 강세 위치가 헷갈리는 단어, 긴 문장의 인토네이션을 익힐 때 유용하다.

## 적합한 입력

- 강세/발음이 까다로운 단어 (예: comfortable, photograph/photographer/photographic 처럼 강세가 이동하는 단어 패밀리)
- 암기하려는 문장 전체 (섀도잉 대상)
- 라임/리듬을 붙이면 외우기 좋은 단어 묶음

## 절차 (단어)

1. **강세 표시**: 강세가 있는 음절을 대문자나 ' 표시로 강조한다.
2. **리듬화**: 음절을 박자에 맞춰 끊어 읽는 방법을 제시한다 (예: "com-FOR-ta-ble" → "콤 / 포 / 터 / 블", 강박 위치 표시).
3. (선택) **라임/챈트**: 비슷한 발음의 다른 단어와 운율을 맞춘 짧은 챈트를 만든다.

## 절차 (문장)

1. **인토네이션 표시**: 문장에서 강조되는 핵심어(주로 내용어: 명사/동사/형용사/부사)에 강세 표시를 한다. 기능어(관사, 전치사, 대명사)는 약하게 발음됨을 표시한다.
2. **연결 발음(linking) 안내**: 자연스럽게 이어지는 부분(연음)을 표시한다.
3. **섀도잉 가이드**: "이 문장을 3번 듣고, 강세 박자에 맞춰 따라 말해보세요" 형태의 연습 지시를 제공한다.

## 출력 형식 (단어)

```
**음성·리듬 연상**
강세: <단어를 강세 표시와 함께>
리듬: <음절을 박자로 끊은 표현>
```

## 출력 형식 (문장)

```
**음성·리듬 연상**
강세 표시: <문장을 강세(대문자)와 약화(소문자/괄호)로 표시>
연음 포인트: <연음되는 부분 1~2곳>
섀도잉 가이드: "<연습 지시 문구>"
```

## 예시 (단어)

- **comfortable** /ˈkʌmftəbl/
  - 강세: COM-fter-ble (1음절에 강세, "컴퍼터블"이 아니라 "컴프터블"에 가깝게 3음절로 발음)
  - 리듬: "컴 / 프터 / 블" — 박자 강-약-약

## 예시 (문장)

문장: "I should have called you earlier."

```
**음성·리듬 연상**
강세 표시: i should-of CALLED you EARL-ier
연음 포인트: "should have" → "슈도브(/ʃʊdəv/)"로 연음, "called you" → "콜쥬"
섀도잉 가이드: "원어민 음성을 3번 들으며 강세가 들어간 CALLED와 EARLIER에 손뼉을 치듯 박자를 맞춰 따라 말해보세요."
```

## 주의

- 강세/연음 정보는 확실하지 않으면 일반적인 패턴(2음절 이상 단어는 보통 어근에 강세 등)을 기준으로 추정하되, 추정임을 밝힌다.
