# 키워드 연상법 (Keyword / Linkword Method)

## 개요

영단어의 발음에서 한국어로 비슷하게 들리는 소리(키워드)를 찾아내고, 그 키워드와 단어의 뜻을 하나의 장면으로 연결해 기억한다. 발음(소리)과 의미(이미지)를 동시에 인코딩하므로 인출 경로가 두 개가 된다 (이중부호화).

## 적합한 입력

- 구체적 명사, 형용사 (이미지화 쉬운 단어)
- 발음이 독특하거나 한국어 단어와 부분적으로 비슷하게 들리는 단어
- 추상어/기능어에는 비효율적 — 그런 경우 mnemonic-context 또는 mnemonic-storytelling을 우선 고려

## 절차

1. **입력 확인**: 단어, 발음(IPA 또는 한글 표기), 뜻을 확인한다. 뜻을 모르면 알고 있는 지식으로 제공하거나 사용자에게 묻는다.
2. **소리 매칭**: 단어 발음 전체 또는 일부(특히 강세 음절)와 비슷하게 들리는 한국어 단어/구를 1~2개 찾는다.
   - 전체 단어와 매칭이 어려우면 음절 단위로 쪼개서 매칭해도 된다.
   - 억지스러운 매칭보다 자연스럽게 떠오르는 소리를 우선한다.
3. **장면 만들기**: 매칭된 한국어 키워드와 영단어의 뜻을 하나의 짧고 생생한(가능하면 우스꽝스럽거나 과장된) 장면/문장으로 연결한다.
   - 감각적 디테일(색, 크기, 소리, 동작)을 넣으면 효과가 커진다.
4. **검증**: 만든 연상이 실제로 단어의 뜻을 떠올리는 데 도움이 되는지 한 번 더 확인한다. 너무 복잡하면 단순화한다.

## 출력 형식

```
**키워드 연상법**
- 발음 매칭: "<영단어 발음>" → "<비슷한 한국어 소리>"
- 연상 장면: <한 문장의 생생한 장면 묘사>
```

## 예시

- **mirage** /mɪˈrɑːʒ/ (신기루)
  - 발음 매칭: "미라지" → "미라(mummy) + 지(地, 땅)"
  - 연상 장면: 사막 한가운데, 붕대를 감은 미라가 걸어오는 게 보였는데 가까이 가니 아무것도 없었다 — 그건 신기루(mirage)였다.

- **gloomy** /ˈɡluːmi/ (우울한, 어두침침한)
  - 발음 매칭: "글루미" → "그루(growth, 그루터기) + 미(美)"
  - 연상 장면: 베어진 그루터기만 남은 어두운 숲, 비 오는 날의 그 음울한(gloomy) 분위기.

## 주의

- 한 단어에 너무 많은 매칭을 만들지 않는다 (1개가 가장 강력함).
- 사용자가 직접 떠올린 연상이 더 잘 기억되므로, 가능하면 사용자에게 "이 매칭이 자연스럽게 느껴지나요?"라고 확인을 유도해도 좋다.
